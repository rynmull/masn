# Masn AAC Platform User Guide

Welcome to Masn AAC! This guide will help you get started with our augmentative and alternative communication (AAC) platform. We've designed Masn to be intuitive and accessible for both users and caregivers.

## Installation

1. **Download the App**
   - iOS: Visit the App Store and search for "Masn AAC"
   - Android: Find us on Google Play Store
   - Desktop: Download from masn.app/download

2. **First-Time Setup**
   - Launch the app
   - Create an account or sign in
   - Follow the quick setup wizard to:
     - Choose your initial vocabulary set
     - Set up basic preferences
     - Configure voice settings

## Basic Usage

### Home Screen Navigation
- Large, clear tiles show common words and phrases
- Swipe left/right to switch categories
- Press and hold any tile to edit or customize it
- Quick access bar at bottom for frequently used phrases

### Speaking
1. Tap words or phrases to build your message
2. Message appears in the preview bar at the top
3. Hit the speak button (📢) to voice your message
4. Use the clear button (🗑️) to start over

### Customizing Your Board
1. Enter Edit Mode (tap the pencil icon)
2. Tap any tile to:
   - Change the text
   - Add/change picture
   - Adjust size
   - Move location
3. Save changes when done

## Caregiver Mode

Caregiver Mode provides additional controls and monitoring capabilities while maintaining user dignity and independence.

### Accessing Caregiver Mode
1. Tap the profile icon
2. Select "Switch to Caregiver Mode"
3. Enter your caregiver PIN

### Key Features
- **Usage Reports**: Track communication patterns and frequently used phrases
- **Remote Support**: Help users from your device
- **Vocabulary Management**: Add/remove words and customize categories
- **Backup Settings**: Save user preferences and customizations

## Voice and Tone Settings

### Voice Customization
- Choose from multiple natural-sounding voices
- Adjust speaking:
  - Speed
  - Pitch
  - Volume
  - Gender
  - Accent

### Pronunciation Helper
- Add custom pronunciations for names or specific words
- Record your own voice for personal phrases
- Fine-tune emphasis for better clarity

## Accessibility Features

### Visual Supports
- High contrast mode
- Adjustable text size
- Color coding options
- Symbol support levels
- Custom visual themes

### Input Methods
- Touch support
- Switch access
- Eye tracking compatibility
- Head pointer support
- Keyboard shortcuts

### Motor Assistance
- Adjustable hold duration
- Touch area size options
- Prediction and auto-complete
- Scanning modes for limited mobility

## Cloud Sync

### Automatic Backup
- All settings and customizations backed up automatically
- Sync across multiple devices
- End-to-end encryption for privacy

### Managing Your Data
1. View sync status in Settings > Cloud
2. Choose what to sync:
   - Vocabulary sets
   - Custom phrases
   - Settings
   - Usage data

### Offline Usage
- Full functionality maintained offline
- Changes sync when connection restored
- Manual sync option available
